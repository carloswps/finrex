# Finrex

Sistema de gestão financeira desenvolvido com foco em organização de gastos e controle financeiro.

## Tecnologias

- Next.js
- TypeScript
- React
- TailwindCSS
- PostgreSQL
- Docker

## Funcionalidades

- Cadastro de usuários
- Controle de despesas
- Histórico financeiro
- Dashboard financeiro
- Integração com API

## Como executar

```bash
npm install
npm run dev
```

## Objetivo

Praticar desenvolvimento fullstack, APIs REST, banco de dados e arquitetura de aplicações.
