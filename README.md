# 🧠 Finrex – API (Back-end)

Este repositório contém o front-end do projeto Finrex, desenvolvido com Next.js e TypeScript, visando proporcionar uma interface moderna e responsiva para o controle de finanças pessoais.
---

## 🔧 Tecnologias

- Next.js
- TypeScript
- React
- Tailwind CSS
